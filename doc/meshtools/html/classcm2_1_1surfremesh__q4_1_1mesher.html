Detailed Description

3-D surface all-quad or quad-dominant remesher.

This component improves the quality of a 3-D surface mesh by remeshing the surface.

The algorithm works as follow:

  • Initial mesh cleaning: it merges nodes that are too close to each other, fixes some topological errors.
    This step is useful when the initial mesh has degenerated elements such as in many STL meshes.
  • Mesh partitioning: it finds group of connected faces (patches) for which the maximum angle between two adjacent elements is lower than a prescribed value (settings_type::patch_angle_tolerance).
    The boundaries lines between the patches are called skeleton lines.
    Each patch can be unfolded to the Z = 0 plane with an induced strain lower than a prescribed value (settings_type::strain_tolerance).
    The algorithm also makes sure that when unfolded, there is no self-intersection within each patch.
  • Metrics field computation: the metrics are computed on each node using the user-specified metric vector (data_type::metrics) and the distances between neighbouring skeleton edges.
    Values are then bounded between settings_type::min_h and settings_type::max_h.
  • Skeleton lines remeshing: generates skeleton edges so that the length of the edges fits the metric field best.
    This gives the remeshed skeleton lines.
  • Patches remeshing: the patches bounded by the remeshed skeleton lines can now be remeshed.
    The nodes of the new mesh are projected back onto the initial mesh.
  • Trans-patch optimization: optimizations are carried out on the new global mesh (edge swapping and node merging using the settings_type::optim_tolerance).
Note
The new mesh is usually not geometrically coincident with the initial mesh.
The nodes of the new mesh are located onto the initial mesh but the nodes of the initial mesh may be a distance away from the new mesh.
An approximate measure of this chordal error between the two meshes is given upon exit by the parameter data_type::max_error_distance.
The input mesh can be hybrid (mixed quads + triangles), all-quad or all-triangle.
Thread safe.
