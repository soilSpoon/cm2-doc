Detailed Description

3-D surface all-quad or quad-dominant mesher on parametric models.

Parametric surface mesher (direct meshing) using an extern CAD kernel.

Runs on BREP, STEP and IGES files.

CAD model made of 57 surfaces (colorized).
Note
Meshing failures can be caused by pathological surfaces (for instance with foldings and self-intersections).
In these cases, check your model and fix if needed.
At present, the mesh-size gradation control doesn't work between curves nor surfaces (only within each curve and each surface).
Bad curve definition can cause accumulation points.
A post-meshing correction step may be needed (from simple node merging to local meshings).
Non-connected surfaces (for instance surfaces with gaps between them) give unconnected meshes (this tool doesn't fix these kinds of topological error).
Examples
tmsh3ds01.cpp.
