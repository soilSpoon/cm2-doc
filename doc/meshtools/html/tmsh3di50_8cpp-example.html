/*
Intersection between 2, 3 or 4 triangle meshes with cm2::intersect_t3::mesher.
Intersections are managed between several meshes (up to four) and between each mesh.
In case of intersection between two faces, the intersecting points are localized on the face of the lowest ID.
So, first faces of the first mesh have the highest priority, then last faces of the first mesh, down to the
last faces of the last mesh.
\param[in] filename0 File with first mesh (can be STL, Neutral, FEMAP, WaveFront OBJ or standard CM2).
\param[in] filename1 File with second mesh (can be STL, Neutral, FEMAP, WaveFront OBJ or standard CM2).
\param[in] filename2 File with third mesh (can be STL, Neutral, FEMAP, WaveFront OBJ or standard CM2).
\param[in] filename3 File with fourth mesh (can be STL, Neutral, FEMAP, WaveFront OBJ or standard CM2).
\param[in] tol Tolerance for the intersections.
Distances below \p tol are considered as null => intersection.
The intersection point is localized on the highest priority face.
\param[in] analysis_code 0 : inter-analysis (the default).
1 : intra analysis.
2+ : full analysis.
*/
void

double tol, unsigned analysis_code)
