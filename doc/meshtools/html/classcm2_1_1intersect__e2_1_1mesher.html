Detailed Description

Edge-mesh intersector.

This tool finds out all the edge-edge intersections/overlappings and corrects them by splitting the intersecting edges (2-D or 3-D).

It may be needed prior to a 2-D plane meshing (by triamesh_iso::mesher or quadmesh_iso::mesher) in order to fix all the intersection problems in a boundary (2-node edges) mesh.

Two edges are considered as intersecting each other when their distance is within a given tolerance parameter (field of settings_type).

When two (non-overlapping) edges are intersecting each other, a new node is created along the first edge (lowest ID in the data_type::connectM matrix) and these edges are split using this new node.

Warning
The tolerance parameter is used to find the initial intersections.
Even in case of success (all initial intersections fixed), new nodes in final split edges may be closer than this tolerance parameter.
Note
The new split faces may need a remeshing (see meshtools1d::remesh_lines) to improve elements length.
Thread safe.
See also
meshtools1d::remesh_lines, meshtools2d::boolean_ops, triamesh_iso::mesher, quadmesh_iso::mesher.

tmsh2di07.cpp, and tmsh2di50.cpp.
