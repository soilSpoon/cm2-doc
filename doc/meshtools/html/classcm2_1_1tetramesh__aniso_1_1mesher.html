Detailed Description

Unstructured anisotropic tetrahedral mesher.

Instances of this class generate tetrahedron meshes on 3-D domains defined by boundary triangle meshes and possibly some embedded nodes, edges and faces.

They can also be used as optimizers of already existing tetrahedron meshes.

All the data to be passed to the mesher are gathered in a single structure of type tetramesh_aniso::mesher::data_type.

Note
Upon entry, the boundary faces in data_type::connectB need to be oriented the same way when there are several subdomains to be meshed simultaneously or when internal cavities are present.
Thread safe.
