CM2 MeshTools SDK Documentation

CM2 MeshTools is a C++ library of high-performance classes and functions to create and manipulate meshes.
CM2 MeshTools is made up of 13 mesh generators and four auxiliary packages, all based on the CM2 Math1 SDK library.


Mesh Generators

CM2 MeshTools contains seven unstructured mesh generators: triangular, quadrangular, tetrahedral meshers (isotropic and anisotropic) and an hexahedral mesher (isotropic).
All these meshers have very similar APIs and are OMP-parallelized.

Plane Mesh Generators

CM2 TriaMesh Iso

CM2 TriaMesh Iso is a 2-D unstructured triangle mesher.
Domains can be of any plane shape (but on the Z=0 plane): they can be convex or not, with holes, with hard nodes or hard (embedded) edges inside.
CM2 TriaMesh Iso is a constrained mesher: it forces all initial hard edges and nodes (outer or embedded) to be present in the final mesh (no other boundary edge added).
It supports a "background mesh" option for mesh adaptivity.

CM2 QuadMesh Iso

CM2 QuadMesh Iso is a 2-D unstructured quadrangle mesher (all-quads or quad-dominant).
Just like the previous triangle mesher, domains can be of any shape (but on the Z=0 plane). They can be convex or not, with holes, with hard nodes or hard edges inside.
In all-quad mode, this mesher needs all boundary lines to be meshed with an even number of edges. This can be achieve using the force_even flag in 1D-mesh creation functions (such as cm2::meshtools1d::mesh_straight).
CM2 QuadMesh Iso is also a constrained mesher. It supports also the "background mesh" option for mesh adaptivity.

CM2 TriaMesh Aniso

CM2 TriaMesh Aniso is a 2-D unstructured anisotropic triangle mesher.
This mesher has the very same options and features as its isotropic counterpart CM2 TriaMesh Iso.
The difference is that the isotropic mesher tends to produce equilateral triangles as much as possible whereas this anisotropic mesher generates stretched elements according to a user-defined 2-D anisotropic metric field.
In a sense, CM2 TriaMesh Aniso produces also "equilateral" triangles but with respect to the local anisotropic metric field.
This mesher can be used as a 3-D surface mesher of any parametric surface with the cm2::meshtools2d::mesh_surface_param.
It supports also the "background mesh" option for mesh adaptivity.

CM2 QuadMesh Aniso

CM2 QuadMesh Aniso is a 2-D unstructured anisotropic quadrangle mesher (all-quads or quad-dominant).
This mesher has the very same options and features as CM2 QuadMesh Iso but with an anisotropic metric field similar to CM2 TriaMesh Aniso.
The isotropic mesher tends to produce squares whereas CM2 QuadMesh Aniso generates stretched elements (rectangles) according to a user-defined 2-D anisotropic metric field.
This mesher can be used as a 3-D surface mesher of any parametric surface with the cm2::meshtools2d::mesh_surface_param.
It supports also the "background mesh" option for mesh adaptivity.


Surface Mesh Generators

CM2 SurfMesh T3/Q4

CM2 SurfMesh T3 (resp. CM2 SurfMesh Q4) generates 3-D surface meshes onto parametric surfaces (BREP, STEP, IGES or directly a TopoDS_Shape OCCT object).
It's based on CM2 TriaMesh Aniso (resp. CM2 QuadMesh Aniso) for the meshers and OpenCascade for the CAD kernel.
Anisotropic surface meshes can be generated to minimize the number of elements while respecting a maximal chordal error between the CAD surfaces and the elements.
Quadratic (T6, Q8, Q9) elements can be generated (curved with high-order nodes onto the surfaces or straight).

CM2 SurfRemesh T3/Q4

CM2 SurfRemesh T3 (resp. CM2 SurfRemesh Q4) is a surface remesher that regenerates new triangle (resp. quad-dominant or all-quad) meshes on initial meshes (can be STL triangle meshes).
They can decimate, refine or simply optimize any surface mesh.
You can "freeze" a set of elements/edges/nodes. You can specific mesh density on any initial node.


Solid Mesh Generators

CM2 TetraMesh Iso

CM2 TetraMesh Iso is a 3-D unstructured tetrahedron mesher.
Domains can be of any shape: convex or not, with cavities, hard nodes, hard edges or hard (embedded) faces inside.
CM2 TetraMesh Iso is a constrained mesher, i.e. it forces all initial boundary edges and faces to remain in the solid mesh.
Topological errors (gaps, overlapping elements) in the boundary mesh can be repaired to some extent (CM2 Intersect T3 is designed to repair more serious intersections and overlapping problems).
CM2 TetraMesh Iso also supports the "background mesh" option for mesh adaptivity.

CM2 TetraMesh Aniso

The CM2 TetraMesh Aniso package provides a 3-D unstructured anisotropic tetrahedron mesher.
This mesher has the very same options and features as CM2 TetraMesh Iso but with an 3-D anisotropic metric field. It generates stretched tetrahedrons according to the local metric.
CM2 TetraMesh Iso tends to produce equilateral tetrahedrons whereas CM2 TetraMesh Aniso generates stretched elements according to the user-defined 3-D anisotropic metric field.
This mesher produces "equilateral" tetrahedrons with respect to the local anisotropic metric field. It supports also the "background mesh" option for mesh adaptivity.

CM2 HexaMesh Iso

The CM2 HexaMesh Iso package provides an unstructured 3-D hex-dominant mesher.
From all-quad, all-triangle or mixed quad-triangle boundary meshes, CM2 HexaMesh Iso generates 3-D hex-dominant meshes (mixed meshes with hexahedrons, wedges, pyramids and tetrahedrons).
Just as CM2 TetraMesh Iso, CM2 HexaMesh Iso accepts domains of any shape: convex or not, with cavities, hard nodes, hard edges or hard faces inside (embedded faces).
Boundary triangle faces and flat boundary quadrangle faces are enforced in the final mesh. However, a few warped boundary quads may be split into two triangles.
It supports also the "background mesh" option for mesh adaptivity.


Specialized Tools

CM2 Intersect E2

The CM2 Intersect E2 package provides a tool for fixing intersecting/overlapping edge meshes (2-node edges).
From a set of edges (plane or 3D), this tool detects and fixes all overlaps and intersections within a given tolerance adding new nodes and new connections.
Though the remaining mesh is pathology-free, a remeshing (using cm2::meshtools1d::mesh_line or cm2::meshtools1d::mesh_straight) of the new elements in the intersecting areas may be needed to improve their quality.

CM2 Intersect T3

The CM2 Intersect T3 package provides a tool for fixing intersecting/overlapping surface triangle meshes (3-node faces).
Similar to CM2 Intersect E2 but for surface triangles.
From a set of triangles (plane or 3D), this tool detects and fixes all overlaps and intersections within a given tolerance adding new nodes and new connections.
Though the remaining mesh is pathology-free, a remeshing (using CM2 SurfRemesh T3) of the new elements in the intersecting areas may be required to improve their quality.

CM2 Layers 2D

CM2 Layers 2D is a special mesh generator for structured 2-D boundary layers.
From a boundary mesh (made of 2-node edge elements), this tool generates thin elements inwards (all-quads, all-triangles or mixed).
Colliding layers are carefully treated. Element-dependent widths and progressive ratio can be specified.
The remaining inner domain can be filled with any 2-D mesh generator of CM2 MeshTools (CM2 TriaMesh Iso, CM2 QuadMesh Iso, CM2 TriaMesh Aniso, CM2 QuadMesh Aniso) whatever the type of the elements in the boundary layers.

General Purpose Auxiliary Packages

meshtools

The cm2::meshtools functions are general meshes- and points- manipulation functions (1-D, 2-D, 3-D): translation, rotation, copy, merge, extrusion, finding the neighbours of elements, reorienting elements, splitting elements...

meshtools1d

The cm2::meshtools1d functions are specific to line meshes: creation of line meshes by point extrusion, mesh on broken lines, arcs or splines, conversion of linear elements into higher degree elements...

meshtools2d

The cm2::meshtools2d functions are specific to plane/surface meshes: creation of plane/surface structured meshes by 1-D mesh extrusion, Coons meshes, conversion of linear elements into higher degree elements, swapping of 2-D elements, reorientation of 2-D elements...

meshtools3d

The cm2::meshtools3d functions are specific to solid meshes: creation of solid structured meshes by 2-D mesh extrusion, conversion of linear elements into higher degree elements, swapping of 3-D elements...



Compatibility with other CM2 components

CM2 MeshTools is based on the CM2 Math1 SDK .
It can also easily be used in conjunction with the finite element engines in the CM2 FEM SDK .
