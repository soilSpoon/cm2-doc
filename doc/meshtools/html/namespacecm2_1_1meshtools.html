A "sub-domain" is a subset of connected elements.

Two elements are in the same subdomain when there is at least one element-path between them crossing no boundary.

indicating that elements 0, 1 and 4 are in subdomain #0 and elements 2 and 3 are in subdomain #1.

[in]ASCII_flagASCII / Binary flag. Default = true.
Returns
Error code. Negative value in case of error.

Warning
In ASCII mode, the coordinates are written with double precision (i.e. "%lf" format), loosing no precision.
Whereas, the STL binary mode supporting only single precision float values, the coordinates are converted (double to float conversions) prior to writing.
This can cause some accuracy issues when reading back a STL binary file.
Precondition
pos.rows() = 3 (or empty).

All the values in the connectivity matrix are smaller than pos.cols().
Note
Upon entry the file must be opened in write or append mode (text or binary according to ASCII_flag).
