Detailed Description

Triangle-mesh intersector.

This tool finds out all the face-face intersections/overlappings and corrects them by splitting the intersecting faces.

It may be needed prior to a 3-D meshing (by tetramesh_iso::mesher) in order to fix all the intersection problems in a boundary (3-node triangles) mesh.

Two faces are considered as intersecting each other when their distance is within a given tolerance parameter (field of settings_type).

When two faces are intersecting each other, new nodes are created along the first face (lowest ID in the data_type::connectM matrix) and these faces are split using these new nodes.

Warning
The tolerance parameter is used to find the initial intersections.
Even in case of success (all initial intersections fixed), new nodes in final split faces may be closer than this tolerance parameter.
Note
The new split faces may need a remeshing (see surfremesh_t3::mesher) to improve elements quality.
Thread safe.
See also
surfremesh_t3::mesher, surfremesh_q4::mesher, meshtools3d::boolean_ops, tetramesh_iso::mesher.

tmsh3di01.cpp, and tmsh3di50.cpp.
