Multiplies a matrix by a vector, with scaling.

The matrix can be row- or column-oriented (defined by the orientation_tag), symmetric or not (shape_tag), blocked or not (blocking_tag), with separate diagonal or not (diagonal_storage_tag).

   y += a * A % x
   y[i] += a * A(i,j) % x[j], sum on j, for i = 0 .. A.rows()-1
Precondition
A.cols() == x.size()
A.rows() == y.size()
Warning
To get y = a * A % x , don't forget to set y to zero prior to the multiplication.

Symmetric rank-2 update, with scaling.

Adds the symmetric rank-2 of two vectors to a matrix, with scaling.

   A(i,j) += a * (x[i]*y[j] + x[j]*y[i])
Note
2 * a * x[i]*y[i] is added on the diagonal.

x.size() <= A.rows()
x.size() <= A.col()
y.size() <= A.rows()
y.size() <= A.cols()
Warning
