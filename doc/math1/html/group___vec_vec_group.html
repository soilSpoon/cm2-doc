◆ insert()

template<class Vec1, class VecI, class Vec>
bool cm2::vecvec::insert ( const Vec1 & V1,
const VecI & V_indices,
Vec & V )

Inserts some values at specific positions in a vector.

Parameters
[in]V1The values to insert.
[in]V_indicesThe indices in the final vector V of the values V1 to insert.
[in,out]VThe vector.
Precondition
Values in V_indices must be sorted ascendingly and unique.
All the values in V_indices are smaller than V.size() + V1.size() upon entry.
V1.size() == V_indices.size()
Postcondition
Upon exit, the size of vector V is increased by the size of V1.
Note
The indices are considered to be in the final vector (i.e. the vector consecutively extended by the insertion of the previous values).
The indices should be the same as for vecvec::remove below.
Only for dynamic-size vectors.
