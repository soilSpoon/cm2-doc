Copy-add between elements in an array (vector or matrix).

When the array is a vector, this is equivalent to: 

   A[i2] = a * (A[i0] + A[i0])

When the array is a matrix, this is equivalent to: 

   A.col(i2) = a * (A.col(i0) + A.col(i1))

Interpolation between elements in an array (vector or matrix).

When the array is a vector, this is equivalent to: 

   A[i2] = A[i0] + a * (A[i1] - A[i0])

When the array is a matrix, this is equivalent to: 

   A.col(i2) = A.col(i0) + a * (A.col(i1) - A.col(i0))
