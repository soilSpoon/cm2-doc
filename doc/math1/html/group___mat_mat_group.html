Appends a matrix to another one.

If the target matrix M2 is empty upon entry, it is resized to the dimensions of the source matrix M1.

Parameters
[in]M1The source matrix to be appended to M2.
[in,out]M2The target matrix.
Returns
Returns true if the operation succeed. False in case of memory allocation error.
Precondition
M1.rows() = M2.rows() or M1 or M2 is empty.
Warning
The matrices should not overlap.
Note
Only for dynamic-size matrices.

◆ push_back() [2/4]

template<class Mat1Iterator, class Mat2>
bool cm2::matmat::push_back ( Mat1Iterator M1_beg,
Mat1Iterator M1_end,
Mat2 & M2 )

Appends a batch of matrices to another one.

Upon exit, all matrices between M1_beg and M1_end (past-the-end) are successively appended to M2.

This function is more memory efficient than successively appending several source matrices to the target matrix.

The number of rows of all source matrices must be equal and also equal to the number of rows of the target matrix M2, if not empty.

Parameters
[in]M1_begIterator to the first source matrix.
[in]M1_endIterator to the past-the-end source matrix = the next after the last.
[in,out]M2The target matrix.
Returns
Returns true if the operation succeed. False in case of memory allocation error.
Precondition
All source and target matrices must have same number of rows (or be empty).
Note
Only for dynamic-size matrices.

Inserts some columns at specific positions in a matrix.

Parameters
[in]M1The columns to insert.
[in]M_colsThe column indices in the final matrix M of the columns M1 to insert.
[in,out]MThe matrix.
Precondition
Values in M_cols must be sorted ascendingly and unique.
All the values in M_cols are smaller than M.cols() + M1.size() upon entry.
M1.cols() == M_cols.size()
M1.rows() == M.rows()
Postcondition
Upon exit, the number of columns of matrix M is increased by the number of columns of M1.
Note
The indices are considered to be in the final matrix (i.e. the matrix consecutively extended by the insertion of the previous columns).
The indices should be the same as for matmat::remove below.
Only for dynamic-size matrices.

◆ transpose_inverse()

template<class Matrix, class T>

Matrix & invT_A,
T & det )
inline

Generalized transposed inverse of a MxN matrix, with (M,N) = (2,2), (3,2) or (3,3).

Computes the (pseudo) inverse together with the (pseudo) determinant. The definition of the pseudo inverse and pseudo determinant the depends on the dimensions of the matrix :

  • M = N = 2 pseudo inverse = inverse. pseudo det = det.
  • M = N = 3 pseudo inverse = inverse. pseudo det = det.
  • M = 3, N = 2 pseudo inverse = contravariant matrix pseudo det = norm2 of the cross product of the two columns.
                   -1
   B = transpose (A  )
Precondition
Matrix A must be of fixed dimensions 2x2, 3x2 or 3x3.
Postcondition
A % transpose(B) = Id(M)
transpose(A) % B = Id(N)
