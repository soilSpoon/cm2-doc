Detailed Description

template<class T>
class cm2::symmetric_full< T >::segment_type

A segment_type is a view to a column of a matrix.


It is equipped with members such as begin(), end() and size() which make it a candidate for most of the STL algorithms and cm2::vecscal, cm2::vecvec, cm2::matvec functions.
A segment_type can be incremented and decremented to view the next or the previous segment in a matrix.
