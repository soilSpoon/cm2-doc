cm2::vecvec Namespace Reference

Namespace for Vector-Vector template functions. More...

Detailed Description

Namespace for Vector-Vector template functions.

Namespace vecvec is made of two categories of template functions:

  • Basic functions on iterators (can be used with raw pointers) Vector-Vector Basic Template Functions
  • Functions on vector classes (concepts of vector with members begin(), end() and size()) Vector-Vector Template Functions
