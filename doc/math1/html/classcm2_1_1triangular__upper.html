Detailed Description

template<class SymMatrix>
class cm2::triangular_upper< SymMatrix >

Upper triangular part of a symmetric matrix.

This class is almost like its template parameter symmetric matrix class.
It has the same storage.
The main difference is that triangular_upper is not considered as a symmetric matrix.
Its mathematical shape is the same as its storage shape, i.e. a triangular shape.

See also
symmetric_fixed, symmetric_full, triangular_lower.
