template<class IStream, class Vector>
int cm2::vecio::read ( IStream & s,
Vector & V )

Reads a vector from an input stream.

The stream must contain data between brackets like: [6 7 1 10] or 4[6 7 1 10]

Template Parameters
IStreamTemplate input stream class.
VectorTemplate vector class (dense, dynamic-size or fixed-size).
Parameters
[in,out]sThe input stream.
[out]VThe vector.
Returns
The number of values read (when ok) or -1 (IO error).
Note
The Vector class should define sub-types value_type and iterator.
When dynamic-size, the Vector class should be equipped with members clear(), reserve(size_t n), push_back(value_type v).
When fixed-size, the Vector class should be equipped with members iterator begin() and iterator end().
Warning
The Vector class should be full (dense).
