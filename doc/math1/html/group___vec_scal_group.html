Inserts a value at a specific position in a vector.

Parameters
[in]vThe value to insert.
[in]iThe position of insertion in initial vector V.
[in,out]VThe vector.
Returns
Returns true if the operation succeed. False in case of memory allocation error.
Precondition
i <= V.size().
Postcondition
Upon exit, the size of vector V is incremented by one.
Note
Only for dynamic-size vectors.

◆ statistics()

template<class Vec, class T>
void cm2::vecscal::statistics ( const Vec & X,
T & mean,
T & variance )
inline

Statistics on a vector (mean and variance). 

   mean = sum(x[i]) / x.size() 
   variance = sum((x[i]-mean)*(x[i]-mean)) / x.size()
Precondition
The vector must not be empty.
Note
The standard deviation equals to sqrt(variance).
