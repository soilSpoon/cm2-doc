template<class IStream, class Matrix>
int cm2::matio::read ( IStream & s,
Matrix & A )

Reads a full matrix from an input stream.

Stream must contains data formatted like: 

   2x3[6 7  1
       1 10 2]
Template Parameters
IStreamTemplate input stream class.
MatrixTemplate matrix class (dynamic, dense and rectangular).
Parameters
[in,out]sThe input stream.
[out]AThe matrix.
Returns
0 (good) or -1 (IO error).
Note
The Matrix class should be equipped with members clear(), resize(size_t m, size_t n), T& operator()(size_t i, size_t j) and should define sub-type value_type.
Warning
The Matrix class should be full (dense and rectangular).
All terms of the matrix are expected in the stream even the zeros.

Matrix & A )

Reads a transposed full matrix from an input stream.

Stream must contain data formatted like: 

   3x2[6 1
       7 10
       1 2]

This will result in the matrix: 

   2x3[6 7  1
       1 10 2]
Template Parameters

0 (good) or -1 (IO error).
Note

Warning
The Matrix class should be full (dense and rectangular).
All terms of the matrix are expected in the stream even the zeros.

Writes a matrix to an output stream.

The output format is like: 

   2x3[6 7  1
       1 10 2]
Template Parameters
OStreamTemplate output stream class.
MatrixTemplate matrix class.
Parameters
[in,out]sThe output stream.
[in]AThe matrix.
Note
The Matrix class should be equipped with members rows(), cols() and value_type operator()(size_t i, size_t j) const.
Warning
All terms of the matrix are written to the stream even the zeros and the out-of-profile terms (symmetric/sparse matrices).

◆ transpose_write()

template<class Ostream, class Matrix>
void cm2::matio::transpose_write ( Ostream & s,
const Matrix & A )

Writes a transposed matrix to an output stream.

A matrix such as: 

   2x3[6 7  1
       1 10 2]

The output is transposed and will look like: 

   3x2[6 1
       7 10
       1 2]
Template Parameters
OStreamTemplate output stream class.
MatrixTemplate matrix class.
Parameters
[in,out]sThe output stream.
[in]AThe matrix.
Note
The Matrix class should be equipped with members rows(), cols() and T operator()(size_t i, size_t j) const.
Warning
All terms of the matrix are written to the stream even the zeros and the out-of-profile terms (symmetric/sparse matrices).
