Detailed Description

File to include all math1 template functions (cm2::vecscal, cm2::vecvec, cm2::matscal, cm2::matvec and cm2::matmat).

Copyright
1999-2025, Computing Objects, France. www.computing-objects.com

This file must be included to provide access to the template mathematical functions nested in the namespaces cm2::vecscal, cm2::vecvec, cm2::matscal, cm2::matvec and cm2::matmat. The mathematical types in file math1.h are designed to work with these template functions. Many of these template functions can however work well with other types of vector and matrix as soon as they provide basic STL-type members such as begin(), end() and size().
